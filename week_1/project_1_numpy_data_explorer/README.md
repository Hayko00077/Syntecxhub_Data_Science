# NumPy Data Explorer

**Syntecxhub Data Science Internship | Week 1 — Project 1**

## Overview
This project explores the fundamentals of NumPy — one of the core libraries for numerical computing in Python. It covers array creation, indexing, mathematical operations, reshaping, broadcasting, save/load operations, and a performance comparison with Python lists.

## Topics Covered
- Array creation (1D, 2D, zeros, ones, arange, linspace, random)
- Indexing and slicing (basic, boolean, fancy)
- Mathematical operations (element-wise, dot product, sqrt)
- Axis-wise operations (sum, min, max, cumsum along axes)
- Statistical operations (mean, median, std, variance, percentiles)
- Reshaping and broadcasting techniques
- Save/load operations (.npy, .npz, .csv)
- Performance comparison: NumPy vs Python lists
